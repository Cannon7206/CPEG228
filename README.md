# CPEG 228 — Microprocessors Laboratory
---

## Overview

This repository contains lab work for **CPEG 228: Microprocessors Laboratory**, the hands-on companion course to CPEG 208. Labs are implemented in PIC16F84A assembly language using MPLAB X IDE, covering GPIO I/O polling, decision logic, timing, and peripheral interfacing through progressively complex experiments.

## Tools & Environment

- **Microcontroller:** PIC16F84A
- **IDE:** MPLAB X
- **Simulator:** Proteus
- **Language:** PIC16 Assembly (`.asm`)
- **Assembler:** MPASM
